## About

**Craftex** is a dedicated Survival Minecraft server designed to offer a balanced, engaging, and fun survival experience. Built with a focus on community and fair gameplay, Craftex provides a classic survival environment enhanced with features that keep the game exciting for players of all levels.

## Features

- 🌿 **Survival Gameplay:** Experience a pure, challenging survival journey.
- 🤝 **Community Focused:** A welcoming environment for players to collaborate and thrive.
- 🏗️ **Structured Rules:** A fair, enforced ruleset to ensure a pleasant experience for everyone.
- 🛠️ **Balanced Economy:** Systems designed to make resources and progression meaningful.

## Rules

Please adhere to the server rules to maintain a friendly environment:
- No hacking or cheating.
- Respect other players.
- Follow staff instructions.
- No griefing or raiding where prohibited.
