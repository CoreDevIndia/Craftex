Server FAQs.

Our FAQ section answers common questions to help you get started quickly. Find clear information about gameplay, rules, and server features all in one place.

Is Craftex a free-to-play server?
Yes, Craftex is completely free to join and play. You can enjoy all the server features without any cost.
Does Craftex support Bedrock Client?
Yes, Craftex fully supports Bedrock clients. Our server is designed for crossplay, so Bedrock players can join and play seamlessly alongside Java players. Whether you’re on mobile, console, or Windows, you can connect easily and enjoy the same features, events, and community as everyone else.
How do I join the server?
To join Craftex, open Minecraft and go to the Multiplayer menu. Then click Add Server (Java) or Play > Servers (Bedrock) and enter the server address. Once added, select the server and click Join to start playing with the community.
What are the server rules?
Craftex has a set of rules to keep the community safe and fun for everyone, including guidelines on behavior, griefing, and fair play. To view all the rules, please join our Discord and read the full list in the #rules channel.
